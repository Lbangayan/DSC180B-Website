How It Works

Parameters Explained:

  • d (Dataset Size): Number of datapoints in your dataset
  • N (Examples): Number of in-context examples to provide
  • R (Rank): Rank parameter for the underlying model
  • flip_prob (Noise): Probability of label flip (0-1)
